## 🚀 Django REST Blog Platform

A full-stack blog application built using Django and Django REST Framework. This project demonstrates end-to-end backend development, REST API design, and integration with a responsive frontend.

### 🔧 Features
- User authentication (login/signup)
- Create, read, update, delete (CRUD) blog posts
- RESTful API architecture using DRF ViewSets
- Pagination and filtering
- Secure API endpoints
- Scalable backend structure

### 🛠 Tech Stack
- Backend: Python, Django, Django REST Framework
- Frontend: HTML, CSS, JavaScript (Bootstrap)
- Database: MySQL

### 💡 Highlights
- Designed using best practices for REST API development
- Clean and modular project structure
- Production-ready code approach with reusable components
